# Chutes and Ladders


## Getting Started

- install [__Node.js__ platform](https://www.nodejs.org)
``` bash
# clone the repo
git clone https://github.com/jkdamilola/chutes-and-ladders.git --recursive

# change directory into the repo's root folder.
cd repo-root-folder
```
